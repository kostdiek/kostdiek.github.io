Project: Triggers in Movies

For some, certain images, events, or words, can invoke powerful emotions - and maybe ones we'd rather not experience. Or at least not be blindsided by while watching a movie. This simple website/app/algorthim is meant to help with that. Below, you'll find a link to a web-based application where you can search for a movie title and give the website a "trigger" word. On the backend, the app will perform a simple search of the movie's synopsis using IMDb's information for that word. I hope that this helps you in some way.


Link to App Powered by Heroku

Future plans for improvement could contain multiple trigger words, "users" so you don't have to input your triggers each time, and Natural Language Processing to predict if a movie might contain a trigger.
